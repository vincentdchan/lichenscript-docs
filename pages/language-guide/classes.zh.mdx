# Classes

Classes are general-purpose, flexible constructs that become the building blocks of your programâ€™s code.

## Definition Syntax

```js
class Base {

    ty: string

    static new(): Base {
        return Base {
            ty: "Base",
        };
    }

    baseCall() {
        print("base call");
    }

    virtual printName() {
        print("name")
    }

}
```

## Static Field

```js

class Color {

    static const Red = "#FF0000"

}

```

Get access to the static field:

```js
print(Color.Red)
```

## Initialization

```js
conat base = Base.new();
```

Or

```js
conat base = Base { ty: "base" };
```

## Inheritance

```js
class Child extends Base {

    name: string

    static new(): Child {
        return Child {
            ...Base.new(),
            name: "Hello World",
        };
    }

    virtual setName(value: string) {
        this.name = value;
    }

    override printName() {
        print("name: ", this.name)
    }

}
```
